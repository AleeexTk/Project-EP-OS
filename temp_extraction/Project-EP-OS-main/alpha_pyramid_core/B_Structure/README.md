# 🗺️ Z15: Atlas Generator (Alpha Layer)

## Иерархия и Масштабирование

Уровень **Z15** — это первый этап расширения от Вершины.

- **Радиус масштабирования**: $r = 17 - 15 + 1 = 3$.
- По сравнению с Z17, этот уровень позволяет группировать до 3-7 основных доменов (Alpha-секторов).

## Распространение (Propagation)

Z15 принимает «застывшие» законы от Z17 и превращает их в **карту реальности**. Здесь происходит проекция Канона на физические папки.

- **Входящий импульс**: От Z17 через Z16.
- **Исходящий импульс**: К Z13 (интент) через Z14.

## Взаимодействия

- **Atlas Generator**: Сканирует структуру `α`, `β`, `γ` и подтверждает их легитимность.
- **Паритет**: Если папка существует на диске, узел в 3D становится `active`.

## Инфраструктура (Black Door)

- **Z14 Transit Node**: Магистраль связи между Картой (Z15) и Семенами Намерений (Z13).
