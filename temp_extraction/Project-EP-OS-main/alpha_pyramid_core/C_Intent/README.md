# 🌱 Z13: PEAR Seed & Intent (Alpha Layer)

## Иерархия и Масштабирование

Уровень **Z13** — точка входа всех намерений системы.

- **Радиус масштабирования**: $r = 17 - 13 + 1 = 5$.
- Уровень позволяет держать десятки активных "семян" (подзадач) в разных секторах (GOLD, PURPLE и т.д.).

## Распространение (Propagation)

Здесь «Закон» превращается в «Желание». PEAR Seed (Семя) — это упакованный интент пользователя, готовый к прорастанию в Бета-слой.

- **Входящий импульс**: От Z15 через Z14.
- **Исходящий импульс**: К Z9 (Агенты) через **Z10 C-R Gateway**.

## Взаимодействия

- **Intent Capsule**: Хранилище контекста текущей задачи.
- **Memory Seed**: Привязка задачи к долгосрочной памяти.

## Инфраструктура (Black Door)

- **Z12 Alpha Bus**: Внутренняя шина данных Альфа-слоя.
- **Z10 C-R Gateway**: Критический шлюз перехода от Намерений к Действиям.
