## Кураторство PROVOCATEUR

Этот уровень работает по принципу **Диалектического Горнила**. Мы не ищем «среднее арифметическое». Мы ищем истину в столкновении.

- **Scaling Rule**: Чем сложнее задача, тем больше веса у Provocateur (Red) агента.
- **Propagation**: Импульс от агентов (Z9) проходит через «фильтр противоречий».
