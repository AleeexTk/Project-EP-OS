# REFERENCE MODEL v4.0

## Topology
- **Layer Alpha (Z10-Z17)**: The Canon. Governance, policy, and global orchestration.
- **Layer Beta (Z5-Z10)**: The Execution. Tools, agents, buses, and cognitive engines.
- **Layer Gamma (Z1-Z4)**: The Reflection. Audits, logs, and self-health monitoring.

## Interaction Protocol
1. **Intent Arrival**: Handled by Nexus Router (Z16).
2. **Policy Check**: Validated against Policy Bus (Z14).
3. **Execution**: Routed via Z-Bus (B2) to specialized agents.
4. **Observation**: Result archived by Integrity Observer (G2) for future reflection.
