# Протокол Взаимодействия Триады (EvoPyramid Triad Protocol)

Версия: 1.1 (Интеграция с EvoGenesis)

## 1. Концепция Ролей

Взаимодействие в системе регулируется Триадой Эволюции, где каждый агент отвечает за свой аспект жизненного цикла импульса.

### 🟡 Trailblazer (NW - Зеленый)

- **Зона**: Z16 (Efficiency Router).
- **Функция**: Инициатор Маршрута.
- **Задача**: Провести импульс PEAR от Z17 к функциональным ядрам кратчайшим путем.

### 🔴 Provocateur (NE - Темно-Красный Градиент)

- **Зона**: Z14 (Policy/Stability Bus).
- **Функция**: Страж Жизнеспособности.
- **Задача**: Проверка "Хаоса" на соответствие Канону. Стабилизация системы при конфликтах.

### 🟣 Soul (SE - Фиолетовый)

- **Зона**: Z2/Z4 (Reflection Bridge).
- **Функция**: Интегратор Памяти.
- **Задача**: Сохранение опыта (Observer) и регенерация мотивации (Phase VI).

---

## 2. Фазовый Протокол Передачи (Flow)

Процесс обработки импульса (1 = 0 = 7) распределяется по Z-блокам:

1. **МНИФЕСТАЦИЯ (Z17 - 1x1)**: Поступление PEAR.
2. **МАРШРУТИЗАЦИЯ (Z16 - 2x2)**: Trailblazer строит `Execution Trace`.
3. **ОБРАБОТКА (Z15 - 3x3)**: Нечетные ядра выполняют логику.
4. **ВАЛИДАЦИЯ (Z14 - 4x4)**: Provocateur проверяет транзакцию. Если есть риск — эскалация.
5. **ТОНАЛИЗАЦИЯ (Z12 - 6x6)**: Synthesis кристаллизует результат (Integration).
6. **РЕФЛЕКСИЯ (Z2/Z4)**: Soul фиксирует опыт в Ленте Времени.

---

## 3. Геометрическая Расстановка (Systematization)

Модули должны размещаться в рамках своих блоков ответственности:

- **Блок 2x2 (Z16)**: Должен содержать только Nexus-роутеры и диспетчеры.
- **Блок 4x4 (Z14)**: Должен содержать шины политик, авто-корректоры и фильтры безопасности.
- **Блок 6x6 (Z12)**: Должен содержать адаптеры внешних API и сервисы синтеза.

### Принцип Градиента Коспетенции

Внутри каждого блока модули располагаются от центра к периферии:

- **Центральные ячейки**: Критические службы ядра.
- **Периферийные ячейки**: Расширения и экспериментальные функции.
- **Интенсивность цвета**: Чем ближе к центру блока, тем темнее/насыщеннее градиент.
