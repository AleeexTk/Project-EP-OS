# 🎭 Z9: Agent Containers (Beta Layer)

## Иерархия и Масштабирование

Уровень **Z9** — «экватор» Пирамиды, зона максимальной функциональной активности.

- **Радиус масштабирования**: $r = 17 - 9 + 1 = 9$.
- Центр горизонтального масштабирования. Здесь могут параллельно работать десятки специализированных агентов.

## Распространение (Propagation)

Трансформация интента в код, текст или решение. Агенты Триады (Trailblazer, Soul, Provocateur) и Prometheus выполняют коллективную работу.

- **Входящий импульс**: От Z10 (Gateway).
- **Исходящий импульс**: К Z7 (Chaos Bus) для синтеза.

## Взаимодействия

- **4 Core Containers**: Триада + Прометей.
- **Agent Orchestrator**: Управление жизненным циклом сессий.

## Инфраструктура (Black Door)

- **Z8 Agent Bus**: Шина меж-агентского общения.
